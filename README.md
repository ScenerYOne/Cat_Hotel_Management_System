Welcome to the **Cat Hotel Booking System**, a comprehensive platform designed to connect cat owners with premium accommodation services for their pets. This project was developed using the **Scrum Framework**, ensuring agile delivery, continuous improvement, and high-value features.

##  Project Overview

This web application addresses the pain points of cat owners looking for reliable pet boarding. It features a seamless booking experience, room availability checking, and facility viewing. On the backend, it provides hotel staff with tools to manage reservations and check-ins efficiently.

##  Scrum Implementation

Our team adopted the Agile mindset, breaking the development lifecycle into **Sprints**. This allowed us to adapt to changes quickly and deliver functional software incrementally.

###  Scrum Team Roles
* **Product Owner (PO):** Define product vision, manage the Product Backlog, and prioritize features.
* **Scrum Master:** Facilitate Scrum events, remove blockers, and ensure the team follows Scrum principles.
* **Development Team:** Responsible for design, implementation (coding), and testing.

###  Scrum Events
* **Sprint Planning:** Selecting User Stories from the Product Backlog to create the Sprint Backlog.
* **Daily Scrum:** A 15-minute stand-up meeting to synchronize activities and plan for the next 24 hours.
* **Sprint Review:** Demonstrating the working product increment to stakeholders at the end of each sprint.
* **Sprint Retrospective:** Reflecting on the past sprint to improve processes for the next one.

###  Artifacts
* **Product Backlog:** An ordered list of everything that is known to be needed in the product.
* **Sprint Backlog:** The set of items selected for the specific Sprint.
* **Burndown Chart:** Visual tracking of work completed versus time remaining.

##  Key Features

###  For Cat Owners (Users)
* **Search & Filter:** Find rooms based on dates, room types, and price range.
* **Room Details:** View high-quality images and amenities for each room.
* **Booking System:** Real-time reservation and payment simulation.
* **Tracking:** View booking status and booking history.

###  For Admin/Staff
* **Dashboard:** Overview of daily/monthly bookings and revenue.
* **Room Management:** CRUD operations (Create, Read, Update, Delete) for room data.
* **Booking Management:** Approve, decline, or modify reservation requests.
* **Check-in/Check-out:** Manage the status of cat guests.

##  Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/ScenerYOne/scrum_10.git](https://github.com/ScenerYOne/scrum_10.git)
    ```
2.  **Navigate to the directory**
    ```bash
    cd scrum_10
    ```
3.  **Install Dependencies**
    ```bash
    npm install
    # or 'pip install -r requirements.txt' if using Python
    ```
4.  **Environment Configuration**
    * Create a `.env` file and configure your Database URL and API Keys.
5.  **Run the Application**
    ```bash
    npm start
    ```
